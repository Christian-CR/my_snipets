Pseudo Clases

::after

Puede servir para colocar alguna marca o icono después de alguna etiqueta

.external::after {
content: url(external-link.png); padding-left: 5px;
}

::first-letter

Selecciona la primera letra

p::first-letter {
/* aquí hirían los estilos */
}

::placeholder

Selecciona el texto del input que se pone como ejemplo de como se debe llenar el input


::placeholder {
color: #96eb7f; font-family: Lato, sans-serif;
}

::before

Por medio de content o background-image, puede estilizar este mismo elemento y lo coloca en línea respecto a la clase, etiqueta o id al cual se le agregue
